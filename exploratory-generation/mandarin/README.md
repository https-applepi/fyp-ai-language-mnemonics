# Exploratory Mandarin Materials

This folder contains a small number of exploratory Mandarin examples generated during the early multilingual testing phase of the project.

Mandarin was initially included to test whether AI-generated visual keyword mnemonics could work beyond alphabetic European languages. However, this branch of the project was not developed as far as Polish or Spanish because the early outputs revealed important limitations.

## What is included here

This folder contains a small number of early Mandarin examples rather than a full exploratory set.

These examples are still included because they document an important part of the project's development: the point at which Mandarin appeared to be much less suitable for this style of AI-generated visual keyword mnemonic learning.

## Why Mandarin generation was not continued

Generation in Mandarin was not pursued further because the early outputs were less useful and less reliable for the purposes of this project. Common problems included:

- weak or unnatural sound-based mnemonic links
- outputs that relied too heavily on literal text rather than effective visual mnemonic structure
- missing pinyin
- difficulty producing strong visual cues that supported both pronunciation and meaning in a simple, beginner-friendly way

These early limitations made Mandarin a poor fit for the final controlled study, especially compared with Polish.

## Why this folder matters

Although the Mandarin material is limited, it is still an important project artefact. It helps explain why the final study was narrowed to Polish and shows that the language-selection decision was based on observed generation quality rather than convenience alone.

## Model note

All items in this folder were generated with **ChatGPT** during the exploratory generation phase.
