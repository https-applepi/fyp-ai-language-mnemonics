# Experiment Design

This folder documents how the final Polish vocabulary experiment was structured.

The final study used a six-group counterbalanced design with three learning conditions:

1. image + mnemonic explanation
2. explanation only
3. word-translation only

All participants studied the same 10 Polish words, but the condition assigned to each word varied by group. This allowed the project to compare different levels of support while keeping the vocabulary itself constant.

This folder contains:
- the study procedure
- the group allocation and condition mapping
- the participant inclusion criteria

These files are included so that the experimental design can be understood without relying only on the dissertation text.
