# Stimuli List

This file lists the final vocabulary items used in the Polish experiment and briefly describes the mnemonic logic behind each stimulus.

## 1. dom
- **English meaning:** house
- **Mnemonic idea:** The image links *dom* to a dome-like house so that the learner remembers the true meaning, *house*, while the dome shape supports pronunciation recall.
- **Image file:** `dom.png`

## 2. kot
- **English meaning:** cat
- **Mnemonic idea:** The image links *kot* to a cat wearing a coat. The cat should remain the main meaning, while the coat supports the sound cue.
- **Image file:** `kot.png`

## 3. okno
- **English meaning:** window
- **Mnemonic idea:** The image links *okno* to a broken window and an “oh no!” reaction. The window should remain the central meaning.
- **Image file:** `okno.png`

## 4. pies
- **English meaning:** dog
- **Mnemonic idea:** The image links *pies* to a dog and pies. The learner should remember that the true meaning is *dog*, not the food item.
- **Image file:** `pies.png`

## 5. ser
- **English meaning:** cheese
- **Mnemonic idea:** The image links *ser* to “sir” as a sound cue, while making cheese visually dominant.
- **Image file:** `ser.png`

## 6. zima
- **English meaning:** winter
- **Mnemonic idea:** The image links *zima* to a "zebra" as a sound cuee in a winter scene.
- **Image file:** `zima.png`

## 7. sklep
- **English meaning:** shop / store
- **Mnemonic idea:** The image links *sklep* to a shop scene with a memorable supporting sound cue.
- **Image file:** `sklep.png`

## 8. stary
- **English meaning:** old
- **Mnemonic idea:** The image links *stary* to an old-looking star so that the learner remembers *old* as the meaning.
- **Image file:** `stary.png`

## 9. torba
- **English meaning:** bag
- **Mnemonic idea:** The image links *torba* to a torn bag, using the visible tear as a pronunciation cue.
- **Image file:** `torba.png`

## 10. ptak
- **English meaning:** bird
- **Mnemonic idea:** The image links *ptak* to a bird attacking as the sound cue while keeping the bird as the true meaning.
- **Image file:** `ptak.png`
