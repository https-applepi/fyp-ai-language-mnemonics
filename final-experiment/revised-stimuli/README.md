# Revised Stimuli

This folder contains the **revised mnemonic image set** generated after the main experiment.

These revised images were created in response to the delayed quiz results, participant feedback, and the evaluation criteria developed during the project. They were designed as a second, improved version of the original mnemonic materials.

## Important note

These revised stimuli were **not used in the original scored experiment**.

They are included here as a project artefact because they show how the findings of the dissertation were translated into improved mnemonic design.

## Why the revised set was created

The original experiment showed that AI-generated mnemonic images could be helpful, but also revealed several weaknesses:

- some images made the sound cue more memorable than the true meaning
- some cue–meaning relationships were not integrated clearly enough
- some scenes were more abstract or visually confusing than ideal
- some items needed stronger meaning salience and lower clutter

The revised image set was therefore generated to address these issues directly.

## Design goals of the revised set

The revised stimuli aimed to improve:

- **meaning salience**, so that the true meaning is visually dominant
- **cue–meaning balance**, so that the sound cue supports memory without overpowering the meaning
- **single-scene integration**, so that cue and meaning feel like one coherent mnemonic scene
- **simplicity and clarity**, especially for beginner learners
- **overall pedagogical usefulness**, rather than just visual creativity

## Relationship to the dissertation

The revised stimuli correspond to the post-hoc redesign discussed in the dissertation. They function as a proof of concept for a stronger second iteration of the mnemonic-generation pipeline and show how the project moved from open-ended generation toward criteria-aware refinement.

## Model note

These revised stimuli were generated with **ChatGPT** using a more tightly constrained, criteria-aware prompting approach than the original experimental set.
