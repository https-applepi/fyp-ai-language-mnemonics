# Criteria

This folder contains the criteria used to judge the quality of AI-generated mnemonic images.

A key conclusion of the project was that AI-generated mnemonic materials are not automatically useful. Some outputs are memorable and educationally helpful, while others are confusing, misleading, or overly focused on the sound cue rather than the true meaning.

For that reason, this folder contains the rubric used to distinguish stronger mnemonic images from weaker ones.
