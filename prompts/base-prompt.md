# Base Prompt

The following prompt was used during the exploratory generation phase:

Pretend you are trying to teach an introductory language learner the language
(insert language) to an English speaker. Your goal is to help them learn
languages through visual mnemonics. You have to decide 20 words that would
be good to for this new introductory language learner. Once you have decided
these words, you have to create a mnemonic that links the pronunciation of that
word, to one in English and provides a visual mnemonic to help the users. For
example, Prom in polish means ferry, this is the same pronunciation as Prom in
English which is a graduation celebration. Here you could generate an image of
a group of students celebrating Prom on a Ferry. When you create images they
should be clear and concise. The background should be plain white as to not
take away from the main object and the there should be bold black text of the
word on top, in this example, Prom. Do not reuse my example Prom.

## Purpose
This prompt was used to test whether an AI system could:
- choose useful beginner vocabulary items
- generate plausible sound-based keyword links
- create clear visual mnemonic ideas
- produce images that reinforce the true meaning rather than only the sound cue
