# Prompts

This folder contains the main prompts used during the project.

These prompts were an important part of the research process because the project investigated whether AI systems could generate useful mnemonic materials for language learning through prompt-based generation.

This folder includes:
- the main base prompt used for multilingual exploratory generation
- any revised prompts used after weaknesses were identified

The prompts are included as artefacts because prompt design directly affected the quality, clarity, and educational usefulness of the generated outputs.
